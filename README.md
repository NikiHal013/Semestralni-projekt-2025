# Haluzová – Projekt

## 1. Původní nápad

**Název projektu:** Slime Jumper  

**Krátký popis:**  
Platformová hra, ve které je cílem dostat se co nejvýš bez pádu dolů.  
Stylově inspirována hrou *Jump King*.

**Klíčové cíle / funkce:**
- kolize se stěnami a možnost odrazu  
- mechanika „charge-up jump“ (držení skoku = vyšší skok)  
- pohybující se nepřátelé nebo překážky  

---

## 2. Úprava konceptu *(03. 11. 2025)*

**Nový název:** Corebound  

**Koncept:**  
2D mini metroidvania inspirovaná hrami *Hollow Knight*, *Celeste* a *Dead Cells*.

**Inspirace se projevuje především v:**
- art stylu  
- pohybových mechanikách  
- designu levelů / mapy  
- herních mechanikách a typech nepřátel  

### To-Do
- získatelné dovednosti (*obtainable skills*)  
- větší variace nepřátel  
- různé typy útoků  
- „kill zóny“ + pohybující se kill zóny  

### Volitelné *(Optional)*
- hook / grappling mechanika  
- UI rozhraní  
- boss-type nepřítel  

---

## 3. Finální stav projektu

- Projekt je rozdělen na dvě části: **Corebound** a **Ninja Game**
- **Ninja Game** má hotovou herní smyčku (*game loop*) a slouží jako technický základ pro Corebound
- **Corebound** je nedokončená verze, která však obsahuje téměř všechny plánované cíle, funkce a To-Do prvky
